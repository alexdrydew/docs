| Parameters | Type | Description |

`input_values`|**Optional\[Dict\[str, Any\]\]**|<p>Input data for a task. `id`|**Optional\[str\]**|<p>Task ID.</p>
`origin_task_id`|**Optional\[str\]**|<p>ID of the task it was copied from.</p>
